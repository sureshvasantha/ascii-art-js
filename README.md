# ASCII Art Generator

A web-based ASCII art generator built with JavaScript and p5.js. This application converts images into ASCII art by mapping pixel brightness values to ASCII characters.

## Features

- Convert images into ASCII art
- Real-time image processing
- Browser-based interface
- Uses p5.js for image manipulation

## Getting Started

### Prerequisites

No special installation is required. You just need a modern web browser.

### Running the Application

1. Clone this repository
2. Open `index.html` in your web browser
3. Follow the on-screen instructions to create ASCII art

## Project Structure

- `index.html` - Main HTML file
- `ascii_art.js` - Core ASCII art conversion logic
- `p5.js` - p5.js library for image processing

## Usage

1. Open the application in your browser
2. Upload an image or use the webcam
3. The image will be automatically converted to ASCII art
